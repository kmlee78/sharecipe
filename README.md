# Sharecipe

DB Project : Recipe sharing service

## Prerequisite

- Linux/Unix OS
- docker & docker-compose

## Development Process

- build development server

```
make dev-up
```

- start development server

```
make run-server
```

- terminate development server

```
make dev-down
```
